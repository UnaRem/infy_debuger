# 英飞源充电模块 CAN 通讯协议 V1.09 全命令手册

本手册汇总了协议中定义的所有读取（Read）与设置（Set）命令，包括数据结构定义及报文示例。

## 一、 帧标识符 (ID) 组成结构

在阅读命令前，请知晓 CAN ID 的 29 位构成：

- **Bit 28-26**: 错误码（0: 正常, 2: 命令号错, 3: 数据错, 7: 启动中）
- **Bit 25-22**: 设备号（0x0A: 单模块协议, 0x0B: 模块组协议）
- **Bit 21-16**: 命令号（0x01 ~ 0x1F）
- **Bit 15-8**: 目的地址（0x00-0x3B: 模块, 0x3F: 广播, 0xF0-0xF8: 监控）
- **Bit 7-0**: 源地址

## 二、 读取命令列表 (Read Commands)

### 0x01 读取系统电压电流 (浮点数)

- **功能**: 读取全系统或组内总输出。
- **回复内容**: 4字节浮点数电压 + 4字节浮点数总电流。
- **示例**:
  - 监控下发: `02 81 3F F0 00 00 00 00 00 00 00 00` (读取系统信息)
  - 模块回复: `02 81 F0 3F 43 FA 00 00 42 48 00 00` (电压 500V, 电流 50A)

### 0x02 读取模块数量

- **功能**: 获取系统中在线的模块总数。
- **回复内容**: Byte0: 模块数量。
- **示例**:
  - 回复: `02 82 F0 3F 00 00 07 00 00 00 00 00` (系统共有 7 个模块)

### 0x03 读取模块 N 电压电流 (浮点数)

- **功能**: 查询指定地址模块的实时数据。
- **示例**:
  - 监控下发: `02 83 00 F0 00 00 00 00 00 00 00 00` (读 0# 模块)
  - 模块回复: `02 83 F0 00 43 FA 00 00 40 60 00 00` (电压 500V, 电流 3.5A)

### 0x04 读取模块状态 (包含组号/温度/状态位)

- **回复结构**:
  - Byte0-1: 模块组号/保留
  - Byte2: 模块环境温度 (8位有符号数)
  - Byte5: 状态表2 | Byte6: 状态表1 | Byte7: 状态表0
- **示例**:
  - 回复: `02 84 F0 00 00 00 02 00 18 00 40 00` (组号 2, 24℃, Walk-in 使能)

### 0x06 读取模块三相输入电压 (定点数)

- **数据单位**: 0.1V。
- **回复结构**: AB电压高低位, BC电压高低位, CA电压高低位。
- **示例**:
  - 回复: `02 86 F0 00 0F B4 0F A5 0F A0 00 00` (AB: 402.0V, BC: 400.5V, CA: 400.0V)

### 0x08 读取系统电压电流 (定点数 mV/mA)

- **功能**: 同 0x01，但精度为毫伏/毫安。
- **示例**:
  - 回复: `02 88 F0 3F 00 03 0D 40 00 00 13 88` (电压 200V, 电流 5A)

### 0x09 读取模块 N 电压电流 (定点数 mV/mA)

- **功能**: 同 0x03，但精度为毫伏/毫安。

### 0x0A 读取模块限值信息 (定点数)

- **回复结构**:
  - Vomax(2B): 最大电压 (V)
  - Vomin(2B): 最小电压 (V)
  - Iomax(2B): 最大电流 (0.1A)
  - Prate(2B): 额定功率 (10W)
- **示例**:
  - 回复: `02 8A F0 00 02 EE 00 64 01 00 05 DC` (750V, 100V, 25.6A, 15KW)

### 0x0B 读取模块条码 (Barcode)

- **功能**: 获取模块的 12 位十六进制出厂条码。

### 0x0C 读取外部电压与允许电流

- **说明**: 电压单位 0.1V，电流单位 0.1A。
- **示例**:
  - 回复: `02 8C F0 00 13 58 01 66 00 00 00 00` (外压 495.2V, 允许电流 35.8A)

## 三、 设置命令列表 (Set Commands)

### 0x1A 开关机控制

- **数据**: Byte0: 1 表示关机，0 表示开机。
- **示例**:
  - `02 9A 3F F0 01 00 00 00 00 00 00 00` (广播关机)
  - `02 9A 3F F0 00 00 00 00 00 00 00 00` (广播开机)

### 0x1B 设置系统总电压/总电流

- **逻辑**: 设置全系统或组的总能力，电流会自动分配给各模块。
- **数据**: 电压 (mV, 4B) + 总电流 (mA, 4B)。
- **示例**:
  - `02 9B 3F F0 00 04 93 E0 00 00 27 10` (设置 300V, 10A)

### 0x1C 设置模块级电压/电流

- **逻辑**: 可针对单模块、组或广播设置特定的电压电流值。
- **数据**: 电压 (mV, 4B) + 电流 (mA, 4B)。
- **示例**:
  - `02 9C 00 F0 00 03 0D 40 00 00 13 88` (设置 0# 模块为 200V, 5A)

### 0x11/0x8F 综合设置命令 (V1.09 新增)

针对支持该功能的型号，通过子命令号区分：

- **子命令 0x11**: 设置工作模式（0xA0: DCDC, 0xA1: MPPT, 0xA2: 输入恒压）。
- **子命令 0x13**: 设置降噪模式（0xA0: 功率优先, 0xA1: 降噪, 0xA2: 静音）。
- **子命令 0x14**: 设置高低压模式（0xA0: 低压, 0xA1: 高压, 0xA2: 自动）。
- **子命令 0x13-0x01**: 设置液冷温度（下发进/出水口温度及环温）。

### 0x13 设置模块 Walk-in (软启动)

- **数据**: Byte0: 1 使能，0 禁止。

### 0x14 设置绿灯闪烁

- **功能**: 用于现场定位模块。Byte0: 1 闪烁，0 正常。

### 0x16 设置模块组号

- **功能**: 将模块归类到特定控制组。Byte0: 组号 (从 1 开始)。

### 0x19 设置模块休眠

- **功能**: 低负载时让特定模块待机。Byte0: 1 休眠，0 不休眠。

### 0x1F 设置地址分配方式

- **数据**: Byte0: 1 地址拨码方式，0 地址自动分配方式。

## 四、 状态位详表 (Byte5/6/7 解析)

| 状态表               | Bit 位  | 含义 (为 1 时)                            |
| -------------------- | ------- | ----------------------------------------- |
| **状态表 0 (Byte7)** | Bit 0   | 输出短路告警                              |
|                      | Bit 1   | DC 侧处于关机状态                         |
|                      | Bit 2   | 模块故障告警                              |
|                      | Bit 5   | 模块休眠                                  |
| **状态表 1 (Byte6)** | Bit 0   | 模块通信中断告警                          |
|                      | Bit 2   | 模块保护告警                              |
|                      | Bit 4   | 过温告警                                  |
|                      | Bit 5   | 输出过压告警                              |
|                      | Bit 6   | Walk-in 使能                              |
| **状态表 2 (Byte5)** | Bit 0   | 模块 ID 重复                              |
|                      | Bit 1   | 模块处于限功率状态                        |
|                      | Bit 4   | 三相输入缺相告警                          |
|                      | Bit 7   | PFC 侧处于关机状态                        |
| **状态表 3 (Byte4)** | Bit 0-1 | 高低压模式 (01: 自动, 10: 低压, 11: 高压) |

*注：所有命令发送间隔建议在 50ms~200ms 之间，避免低于 20ms 的高速下发导致模块处理不及时。*
