## 2. 命令体系总览

从图片资料可明确识别出以下命令规则：

- `CMD = 0x23`：**读取模块 / 系统信息**
- `CMD = 0x24`：**设置模块 / 系统参数**

命令的数据域采用分层分类方式：

- `Byte0`：大类
- `Byte1`：小类
- `Byte2 ~ Byte7`：参数值或扩展字节

### 2.1 已识别的大类含义

| Byte0 | 含义 |
| --- | --- |
| `0x10` | 系统基本信息 / 系统直流侧信息 |
| `0x11` | 单模块基本信息 / 模块直流侧信息 |
| `0x12` | 风扇、水温等辅助信息 |
| `0x21` | 单模块交流侧信息 / 交流侧控制参数 |
| `0x31` | 调度、功率模式等控制类参数 |
| `0x40` | 系统直流高压侧信息 |
| `0x41` | 双向 DCDC 模块高压侧信息 |
| `0x51` | 电池保护阈值类参数（由演示命令可识别） |

### 2.2 命令作用域说明

图片中对命令下发范围有明确说明：

- **系统命令**：针对整个系统，只能广播发送或按组发送，**不能**按单模块地址发送。
- **组命令**：针对指定组号，设置值会平均分配到组内各模块。
- **模块命令**：针对单个模块。

其中还有两个重要行为规则：

1. **功率 / 电流类设置**会平均分配到系统内或组内各模块。
2. **电压类命令**在实际效果上通常等同于系统命令、组命令或模块命令。

## 5. 典型模式与接线说明

图片中的模式说明可整理为：

- 模块有两种起机方式：**直流起机**、**交流起机**，默认是直流起机。
- 交流起机需要谨慎使用，资料中特别提示：**直流侧不能短路**。
- 直流起机下又有两类调度方式：
  - **交流调度**：控制交流侧的有功 / 无功功率，即 P / Q。
  - **直流调度**：控制直流侧，可细分为恒流、恒功率、恒压模式。

接线示意图中可识别的信息包括：

- 模块并机采用总线型接法。
- CAN 通讯总线两端需要 `120Ω` 终端电阻。
- 资料还给出了两份附件线缆的线序颜色图，但未提供明确针脚功能定义。

## 6. 典型 CAN 开机流程摘要

以下示例来自图片中的“典型模式开机操作命令”。

> 注意：以下报文为资料截图中的示例值，适合作为理解命令关系和开机顺序的参考。实际项目中应以正式协议文档为准。

### 6.1 交流调度并网模式

适用场景：

- 并网模式
- 交流调度
- 控制交流侧功率 `P / Q`

典型步骤如下：

1. **设置 PCS 模块直流起机**
   - 模块广播：`02A43FF0 1127 0000 0000 00A0`
   - 组命令：`02E400F0 1127 0000 0000 00A0`
   - 单模块：`02A400F0 1127 0000 0000 00A0`

2. **设置 PCS 模块交流调度**
   - 模块广播：`02A43FF0 3111 0000 0000 00A1`
   - 组命令：`02E400F0 3111 0000 0000 00A1`
   - 单模块：`02A400F0 3111 0000 0000 00A1`

3. **设置直流电压为 1000V**
   - 系统广播：`02A43FF0 1001 0000 000F 4240`
   - 组命令：`02E400F0 1001 0000 000F 4240`

4. **设置 P 或 Q（示例：P = 50kW）**
   - 可用模块命令 `0x3123`，也可用命令 `0x1003`
   - `0x3123`：
     - 模块广播：`02A43FF0 3123 0000 02FA F080`
     - 组命令：`02E400F0 3123 0000 02FA F080`
     - 单模块：`02A400F0 3123 0000 02FA F080`
   - `0x1003`：
     - 系统广播：`02A43FF0 1003 0000 02FA F080`
     - 组命令：`02E400F0 1003 0000 02FA F080`

5. **设置电池充电结束电压 1000V**
   - 该数值应与步骤 3 保持一致
   - 模块广播：`02A43FF0 5101 0000 000F 4240`
   - 组命令：`02E400F0 5101 0000 000F 4240`
   - 单模块：`02A400F0 5101 0000 000F 4240`

6. **设置电池充电过流点 115A**
   - 系统广播：`02A43FF0 5102 0000 0001 C138`
   - 组命令：`02E400F0 5102 0000 0001 C138`

7. **设置电池放电结束电压 560V**
   - 模块广播：`02A43FF0 5103 0000 0009 27C0`
   - 组命令：`02E400F0 5103 0000 0009 27C0`
   - 单模块：`02A400F0 5103 0000 0009 27C0`

8. **设置电池放电过流点 115A**
   - 系统广播：`02A43FF0 5104 0000 0001 C138`

9. **设置电池 DOD 电压 600V**
   - 模块广播：`02A43FF0 5105 0000 0009 27C0`
   - 组命令：`02E400F0 5105 0000 0009 27C0`
   - 单模块：`02A400F0 5105 0000 0009 27C0`

## 7. 使用上的关键信息

根据图片内容，可以提炼出以下实用结论：

1. **先定模式，再下功率命令。**
   无论是交流调度还是直流调度，示例流程都先完成起机模式、调度模式、电压等基础参数设置，再下发功率和保护参数。

2. **电池保护类参数是开机流程的重要组成部分。**
   充电结束电压、充电过流点、放电结束电压、放电过流点、DOD 电压都在示例中单独配置。

3. **同一目标值在不同命令中可能需要保持一致。**
   例如示例中“直流电压 1000V”和“充电结束电压 1000V”明确要求一致。

4. **系统命令与模块命令并不能完全互换。**
   尤其是系统命令不能以单模块地址发送，这一点在联调时需要特别注意。
